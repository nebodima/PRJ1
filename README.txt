====================================
HelpDesk - Система управления задачами
====================================

ЗАПУСК ПРОЕКТА:

1. Первый раз - установите зависимости:
   Дважды кликните: install.bat

2. Запуск проекта:
   Дважды кликните: start.bat

3. Откроется браузер на http://localhost:5173

====================================

ОСТАНОВКА:
Закройте окна Backend и Frontend

====================================

ТЕХНОЛОГИИ:
- Backend: Node.js + Express + JSON Database
- Frontend: React + Vite + Tailwind CSS

====================================

ФУНКЦИИ:
✓ Создание и редактирование задач
✓ Назначение исполнителей (5 программистов)
✓ Статусы (открыто, в работе, завершено, закрыто)
✓ Приоритеты (низкий, средний, высокий)
✓ Дедлайны с подсветкой просроченных
✓ Срочные задачи с индикатором
✓ Теги для категоризации
✓ Счетчики комментариев, файлов, подзадач
✓ Темная тема в стиле Claude AI
✓ Компактный дизайн для смартфонов
